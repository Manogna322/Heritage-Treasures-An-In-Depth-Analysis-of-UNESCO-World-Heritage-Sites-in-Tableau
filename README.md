# Heritage-Treasures-An-In-Depth-Analysis-of-UNESCO-World-Heritage-Sites-in-Tableau
🌍 Heritage Treasures: An In-Depth Analysis of UNESCO World Heritage Sites

## 📌 Overview
This Tableau project analyzes UNESCO World Heritage Sites to visualize their distribution, categories, endangered sites, and historical trends through interactive dashboards.

## Objectives
- Analyze UNESCO World Heritage Sites worldwide.
- Compare Cultural, Natural, and Mixed heritage sites.
- Identify endangered heritage sites.
- Visualize regional distribution and historical trends.

## Dataset
The dataset includes:
- Site Name
- Country
- Region
- Category
- Area (Hectares)
- Year Inscribed
- Danger Status

## Dashboards
Dashboard 1: Heritage Sites Overview
- Categories by Site Count
- Site Count per Region
- Top 10 Regions by Area
- Year Forecasting
- Region-wise Bubble Chart

Dashboard 2: Danger Sites Overview
- Endangered Regions
- Danger Sites Analysis
- Top 10 Danger Sites

##Tableau Story
    The Tableau Story combines both dashboards into a step-by-step presentation, making it easier to understand the overall analysis and insights.

##Key Insights
-Cultural Heritage Sites have the highest count.
-Europe and North America contain the largest number of UNESCO World Heritage Sites.
-Several heritage sites are currently endangered and require conservation.
-Interactive dashboards simplify complex heritage data for better understanding.
